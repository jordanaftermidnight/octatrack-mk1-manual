Trigs & parameter locks

The Octatrack sequencer's two atomic concepts. Everything else — microtiming, conditional trigs, fill mode, scenes — is built on these.

The trig

A trig is a step in the pattern that fires the track's sound. The trig row at the bottom of the panel (in TRACKS mode) shows 16 trigs at a time; the pattern's full length, up to 64 steps, is paged across the four pages with the LEFT/RIGHT arrow keys.

To add a trig: press a trig key while the corresponding track is selected. The trig key lights, and the trig will fire on that step every time the pattern reaches it. To remove: press the trig key again. To preview a trig (with its locks) without committing it, hold YES + the trig key.

Trig types — pressed in the same way but distinguished by held modifiers:

  • Note trig — the default. Fires the sample at the trig's pitch and parameter values.
  • Trigless lock — held FUNCTION + trig. Records parameter locks at this step but does not re-trigger the sample. Useful for parameter automation that rides over a sustained note.
  • One-shot trig — held YES + trig. Fires once when armed (typically used with the Pickup machine), then disarms itself.
  • Hold trig — long-press a trig in Live Record mode while a sound plays.

The parameter lock

A parameter lock ("p-lock") records a parameter value specifically for one step. When the pattern reaches that step, the parameter takes the locked value; on every other step, it uses the part's base value.

To set a p-lock:

  1. Hold the trig key for the step you want to modify
  2. While holding, turn any data-entry knob (A–F) — the value moves for that step alone
  3. Release the trig key

The trig now has a parameter lock, indicated by a red light on the trig and an inverted parameter label on the LCD when that step is selected. To clear all locks on a step: hold the trig and press NO. To clear locks on a single parameter: hold the trig and press the parameter knob (A–F) corresponding to it.

P-locks work on every parameter the OT exposes — pitch, sample start, slice number, filter cutoff, FX values, even the sample slot itself (a sample slot lock, see chapter 21). This is the OT's signature feature: any single step can sound radically different from any other.

P-lock examples

Recipe — pitch ladder

Place a kick drum sample on every step of a 16-step pattern. Hold step 5 and turn the pitch knob down 5 semitones; hold step 13 and turn it up 7 semitones. The pattern now plays the kick at three distinct pitches across the bar — at the same loudness, same envelope, same FX. One sample, one pattern, three voices.

Recipe — moving filter sweep

With a sustained pad sample on a track, place 16 trigs (one per step) and lock the FX1 filter cutoff at gradually higher values across the pattern. You've drawn a filter envelope without using an LFO — and it's editable on a per-step basis.

Trigless locks

A trigless lock changes parameters without re-triggering the sound. To set: hold FUNCTION + trig key, then turn knobs.

Why this matters: if a sample plays for several steps after its initial trig (long release, sustained loop), p-locking subsequent steps would re-trigger the sample. Trigless locks let you ride parameters over the sustain — a filter sweep on a long pad, a pitch bend on a sustained note, an FX wet/dry change without interrupting the sound.

On the trig row, trigless locks show a different LED state from note trigs. The full LED-state legend is in Appendix A.

The base value vs. the locked value

Every parameter has a base value set on the parameter page (the knob position you see in the LCD when no trig is held). Locks are deltas from this base: at every unlocked step, the parameter uses the base; at locked steps, it uses the lock.

Adjusting the base value changes every unlocked step. This is the fastest way to "shift everything" — turn the base knob, all unlocked steps follow; locked steps stay where you put them.

Tip · the silent-step trick

Want to make a track go silent for a few steps without removing its trigs? P-lock the AMP track-volume to zero on those steps. The trigs still fire, the sample still plays, but at zero level — so when you raise the volume p-lock back up later, the trigs are still there waiting.

The scope reminder

Trigs and p-locks live in the pattern (chapter 6). Switching to a different pattern presents a different set of trigs/p-locks — the part stays the same. This is why two patterns on the same part can sound radically different: the part defines the instrument, but the pattern's trigs and locks define the performance.
