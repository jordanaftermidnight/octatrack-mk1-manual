Static & Flex machines

The two principal playback machines. Static streams from the card; Flex plays from RAM. Almost every Octatrack project uses both.

Static machine

Loads from the Static slot list. Plays the slot's audio file by streaming from the CF card. Constraints: forward playback only at moderate pitch ranges; cannot reverse; cannot pitch-shift below about –12 semitones without artefacts.

Page parameters

PageEncoder ABCDEF
PLAYBACK Slot Pitch Start Length Loop Rate
AMP Atk Hold Rel Vol Bal Xvol

Best for

  • Backing tracks (full songs, stems)
  • Long ambient samples
  • DJ workflows where each track is a complete song
  • Anything over 30 seconds where RAM would be wasted

Flex machine

Loads from the Flex slot list. Audio is in RAM. Supports the full range of OT playback: reverse, pitch shift over a wide range, slice playback, scrubbing, time-stretching.

Page parameters (Flex-specific)

The Flex machine has the same PLAYBACK page layout as Static, plus these additional capabilities:

  • Negative pitch at PTCH –64 = octave down (full speed reverse capable)
  • Reverse via RATE negative values, or via the reverse trig
  • Slice mode via START set to slice number (0–63) when slice grid exists
  • Time-stretch on the TIMESTRETCH page (only on Flex; chapter 30)

The TIMESTRETCH page

Flex tracks have access to a TIMESTRETCH page (accessed by holding FUNCTION + the LFO button). It allows:

  • NORM — pitch and tempo bound together (changing rate changes both)
  • BEAT — tempo follows project tempo, pitch independent (the standard "stretchy" mode)
  • OFF — pitch and rate decoupled, no stretching

BEAT mode is the standard time-stretch behaviour: the sample auto-conforms to project tempo, and you can pitch it independently. Quality is acceptable for moderate stretches; extreme stretches introduce artifacts (which can themselves be musical).

A common pattern: one or two Static tracks for backing material, five or six Flex tracks for everything else. The Master machine on T8 if you need master-bus FX. This is the canonical "live Octatrack" project layout.

Sample slot locks

Both machines support sample slot locks — a parameter lock that swaps the sample slot for a single step. Hold a trig, turn encoder A (the SLOT parameter), and the trig now plays a different slot from the slot list.

This is the Octatrack's "kit-switching per step" capability. With 16 trigs and 16 different slot locks, one track can play 16 different drum sounds across a bar — using only one slot list and one machine.

Tip · slot-lock for kit drumming

Build a slot list of drum hits (kick, snare, hat, clap, perc, etc.). On a single Flex track, place 16 trigs and slot-lock each to a different drum. You've built a one-track drum kit that takes no extra tracks. Combined with pitch p-locks per step you can get surprisingly varied drums from a single track.

Slot setup workflow

  1. FUNCTION + PLAYBACK opens the Setup screen for the active track's machine
  2. Encoder A scrolls through slots; encoder B previews
  3. YES assigns the highlighted slot to the track
  4. EXIT returns to the PLAYBACK page

Slots can be loaded from the Sample Manager if not already assigned. The Setup screen shows the current slot, file name, and estimated memory usage.
