Trig modes

The 16 trig keys do six different jobs depending on the active trig mode. Switching modes is the difference between drawing a sequence, playing a chromatic scale, triggering slices, or muting tracks live.

Switching modes

Hold FUNCTION and press the UP/DOWN arrow keys to cycle through trig modes. The current mode is shown in the LCD title bar.

The six modes:

1. TRACKS mode (default)

Each trig key represents a step in the active track's pattern. Pressing a trig adds or removes that step. This is the "draw the sequence" mode you spend most of your time in.

Sub-behaviours:

  • TRIG alone — toggle step on/off
  • TRIG hold — set p-locks, microtiming, conditions
  • FUNCTION+TRIG — trigless lock
  • YES+TRIG — preview the step (with locks)
  • CUE+TRIG — preview to the CUE bus

2. CHROMATIC mode

The 16 trig keys become a one-octave keyboard. Pressing a trig plays the active sample at a corresponding pitch — trig 1 = root note, trig 2 = root+1 semitone, and so on. Trig 13 is the octave; trigs 14–16 extend three semitones into the next octave.

Useful for:

  • Playing pitched samples (bass, leads, single-cycle waveforms)
  • Live sample-pitching during performance
  • Pattern composition by recording chromatic input into Live Record (chapter 12)

The base pitch is the sample's natural pitch as set on the PLAYBACK page. Octave shifting is via FUNCTION+UP/DOWN within CHROMATIC mode (not the trig-mode switcher — UP/DOWN alone shift octaves once you're in CHROMATIC).

3. SLICES mode

Each trig key plays a different slice of the active sample. Available with Flex machines that have a slice grid defined (chapter 30 — audio editor SLICE page).

Trig 1 plays slice 1, trig 2 plays slice 2, etc. The OT supports up to 64 slices per sample, displayed across four pages (LEFT/RIGHT arrows page through the slice ranges). The current page is shown on the LCD.

Use cases:

  • Beat slicing — chop a drum loop into 8 or 16 slices, play any permutation
  • Vocal cutting — slice a phrase into syllables, retrigger any in any order
  • Build new patterns by slicing and recording slice trigs into Live Record

4. SLOTS mode

Each trig key selects a different sample slot for the active track. Trig 1 = slot 1, trig 2 = slot 2, etc.

SLOTS mode is fast for switching between samples during performance or for recording slot changes into Live Record (giving a track that plays a different sample on each step).

Combined with the sample slot lock (chapter 21), SLOTS mode is how you build pattern-driven sample variation without multiple tracks.

5. QUICK MUTE mode

Each trig key toggles mute for one of the eight tracks. Trigs 1–8 correspond to tracks 1–8; trigs 9–16 are inactive.

QUICK MUTE is a performance mode for live arrangement-building — drop tracks in and out without using FUNCTION+TRACK combinations. It's the fastest way to mute multiple tracks simultaneously (just slap multiple trigs).

A muted track in QUICK MUTE shows on the trig row as a dim LED; unmuted tracks show as bright LEDs.

6. DELAY CONTROL mode

The most performance-oriented mode. Each trig key, when held, momentarily sends the active track's audio through a delay effect at a different feedback amount and time setting per trig. Release the trig and the effect goes back to its base value.

The default mapping:

  • Trigs 1–4: increasing delay time, low feedback
  • Trigs 5–8: medium delay, medium feedback
  • Trigs 9–12: longer delay, more feedback
  • Trigs 13–16: extreme settings (long, near-self-oscillating)

DELAY CONTROL is configured per project; the exact mapping is editable. It's the OT's answer to "instant delay throw" without needing to scene-lock or p-lock — just hold a trig.

Tip · DELAY CONTROL with FX2 reverb

Because DELAY CONTROL momentarily routes through the delay, pairing it with a reverb on FX2 produces "delay-into-reverb" throws that decay naturally when you release the trig. The result is dub-style throws without parameter-lock setup.

Mode persistence

Trig mode is a per-track state. Switching tracks does not change the mode. Switching projects resets to TRACKS mode.

The most common mistake: forgetting you're in CHROMATIC or SLICES mode and trying to add trigs to a pattern. The trigs play sound but don't write to the pattern. Always check the LCD title bar.
