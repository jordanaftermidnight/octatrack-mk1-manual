Appendix C

Glossary

Octatrack-specific terminology. Most terms appear in the official manual; some are community shorthand.

Audio Pool
The set-level folder containing all sample files available to projects within that set. See chapter 7.
Bank
A container holding 16 patterns and 4 parts. A project has 16 banks (A–P). Each bank's patterns share its parts. See chapter 6.
BPM
Beats per minute. The OT's tempo can be project-wide or per-pattern. Per-row override available in arrangements.
Buffer
A recorder buffer — RAM allocated for audio capture on a track. Eight buffers, one per track. See chapter 19.
CHROMATIC mode
Trig mode where the 16 trig keys play the active sample at different chromatic pitches. See chapter 11.
CF card
Compact Flash. The OT's only persistent storage medium. See chapter 6.
Conditional trig
A trig that fires only if a condition is met (probability, previous trig, neighbour, fill, etc.). See chapter 10.
Crossfader
Hardware fader that morphs parameters between Scene A and Scene B. Does not gate audio directly. See chapter 21.
CUE bus
Secondary audio bus, separate from MAIN. Routed to the rear- panel CUE outputs. Useful for headphone pre-listen and Studio Mode. See chapter 8.
DELAY CONTROL mode
Trig mode where each trig key momentarily activates a different delay setting. Performance-oriented. See chapter 11.
FILL mode
Momentary mode that activates FILL-conditional trigs. See chapter 10.
Flex
A machine type. Plays samples loaded into RAM. Supports reversal, pitch, time-stretch, slicing. See chapter 13.
FX1, FX2
Two effects slots per track. FX1 runs first, FX2 second. Catalogues differ slightly. See chapter 18.
LFO designer
Custom-waveform editor for LFOs. 16-point grid with interpolation modes. Hidden behind FUNCTION + LFO. See chapter 17.
Live Record
Real-time pattern entry mode. Activate with REC + PLAY. See chapter 12.
Master machine
A machine type loaded only on T8. Turns T8 into a master-bus processor. See chapter 14.
Memory config
Project-wide RAM allocation between Flex samples and recorder buffers. See chapter 7.
Microtiming
Per-trig timing offset, in 1/23rd-of-a-step increments. See chapter 10.
Neighbor machine
A machine that reads the audio output of the previous track. Used to chain effects across tracks. See chapter 14.
One-shot trig
A trig that fires once, then disarms. Used by Pickup machines and for triggered events. Set with YES + TRIG.
Parameter lock (p-lock)
A parameter value applied to a single step, overriding the part's base value. The OT's signature feature. See chapter 9.
Part
Container for machine assignments, base parameter values, and scene assignments. Four parts per bank. See chapter 6.
Pattern
A sequence of trigs and parameter locks. 1–64 steps. 16 patterns per bank. See chapter 6.
Pickup machine
A live-looper machine type. Records from a chosen source into a buffer, then loops. Master/slave configuration for synced multi-track loops. See chapter 15.
Project
Top-level container holding 16 banks, settings, sample slots, arrangements. See chapter 6.
QUICK MUTE mode
Trig mode where trig keys 1–8 toggle track mutes. See chapter 11.
QREC
Quantise recording. Recording start aligns to a chosen beat division. See chapter 19.
Resampling
Recording the OT's own audio output (MAIN or CUE) into a recorder buffer. See chapter 20.
Scene
A snapshot of parameter values across all tracks. 16 scenes per part. Two scenes assigned to crossfader ends for morphing. See chapter 21.
Scene lock
A parameter explicitly assigned to a scene. The crossfader morphs only scene-locked parameters. See chapter 21.
Set
Top-level folder on the CF card. Holds the Audio Pool and one or more projects. See chapter 6.
Slice
A named region within a sample, addressable as a slice number. Up to 64 slices per sample. See chapter 20.
SLICES mode
Trig mode where each trig key plays a different slice of the active sample. See chapter 11.
SLOT (Sample slot)
An assignment of a sample file to a numbered slot, with trim and loop points. Static slots (128) and Flex slots (128) are separate lists. See chapter 7.
SLOTS mode
Trig mode where each trig key selects a different sample slot. See chapter 11.
Static
A machine type that streams samples from the CF card. RAM- light but limited in playback flexibility. See chapter 13.
Thru machine
A machine that routes external audio input through the track's effects chain. See chapter 14.
TIMESTRETCH
Flex-only feature. Sample tempo and pitch decoupled. Three modes: NORM, BEAT, OFF. See chapter 13.
Trig
A step in the pattern that fires the track's sound. See chapter 9.
Trigless lock
A parameter lock that does not retrigger the sound. Set with FUNCTION + TRIG. See chapter 9.
TurboMIDI
Higher-baud-rate MIDI between two Elektron devices. Reduces clock jitter. Automatic when both ends support it. See chapter 23.
XVOL
Crossfader-controlled volume offset. Used for fader-driven track fade-in/out via scene-locking. See chapter 16.
