Pickup machine

The Octatrack's live looper. Pickup is the trickiest machine to learn — its state model is unfamiliar — but once internalised it's the OT's most powerful performance tool.

What Pickup does

A Pickup machine records audio (from a chosen input or internal source) into its own internal buffer, then plays back what it captured. It's a looper: capture, then loop.

Pickup's distinguishing feature is its tempo-locked behaviour. Unlike a generic looper that captures whatever length you press for, Pickup quantises capture and playback to the project's bar/beat grid. This is what makes it useful in sequence-based performance: the loop you capture fits into the rest of your patterns.

Master and Slave Pickups

Pickup machines come in two flavours, set per track on the Setup page:

  • Master Pickup — the first Pickup loop in a project. Sets the loop length for all subsequent Pickup loops in the same project.
  • Slave Pickup — every Pickup track after the first. Locks to the master's length; allows multi-track looping with synchronised lengths.

Typical use: T1 = Master Pickup, T2/T3 = Slave Pickups. Recording on T1 sets the bar count for everything; T2 and T3 capture loops of identical length, layered.

The state machine

Pickup tracks are always in one of these states:

  • EMPTY — no audio captured. Track is silent.
  • RECORDING — capturing audio into the buffer. Track plays back the audio currently being recorded (overdub possible).
  • PLAYING — looping the captured audio.
  • STOPPED — buffer holds audio but track is silent.
  • OVERDUBBING — playing AND recording over the existing loop (additive layering).

Triggering Pickup

Pickup's trigs are one-shot trigs: they fire once and then disarm. Set with YES + trig key. When the sequencer reaches a one-shot trig on a Pickup track, it triggers the next state transition:

  • EMPTY → RECORDING
  • RECORDING → PLAYING (after the bar count completes)
  • PLAYING → OVERDUBBING (next one-shot)
  • OVERDUBBING → PLAYING
  • STOPPED → PLAYING
  • PLAYING → STOPPED (FUNCTION + one-shot)

The sequence isn't entirely automatic — the next one-shot determines what happens next. With careful pattern design, you can sequence a series of state transitions across a long pattern.

Pickup recording length

The bar count (loop length) is set by the master Pickup track. The options:

  • 1 bar, 2 bars, 4 bars, 8 bars, 16 bars (preset values)
  • "Inf" — record until you press stop, then quantise to the nearest bar

Slave Pickups always capture the master's length. If the master is set to 4 bars, every slave records exactly 4 bars even if you press the trigger early.

Source selection

Pickup records from a chosen source, set on the Setup page:

  • Input AB or CD — record live external audio
  • The MAIN bus (post-mixer) — record the OT's full mix
  • The CUE bus — record a curated sub-mix
  • One specific track's output — capture just that track

Recording from MAIN or CUE creates resampling-style loopers — the OT records its own output back onto a track, layered with whatever you played live.

FUNCTION combinations on Pickup

With a Pickup machine on the active track, several FUNCTION+key combinations gain Pickup-specific behaviour:

  • FUNCTION + MIDI (PART) — opens Pickup-specific options
  • FUNCTION + one-shot trig — clear and re-arm that trig
  • FUNCTION + STOP — stop all Pickup tracks immediately

Performance recipes

Recipe — basic looper

  1. Load a Pickup machine on T1, set as Master, source = Input AB, length = 4 bars.
  2. Plug a microphone or instrument into rear input AB.
  3. Place a one-shot trig on step 1 of T1's pattern.
  4. Press play. T1 records 4 bars of input. After 4 bars it enters PLAYING and loops.
  5. Place another one-shot trig on a later step. When the sequencer hits it, T1 enters OVERDUB. Play more, the layer adds. After 4 bars it's back in PLAYING.

Recipe — multi-track stacking

With T1 as Master and T2/T3 as Slaves, you can layer three independent loops at the same bar length. Each track captures and plays independently — record bass on T1, melody on T2, ambient drone on T3, all at 4 bars, all looping in sync.

Hidden · resampling tricks

Set Pickup's source to MAIN bus and place trigs that capture the OT's own output. Effectively this samples the live mix — including all current FX, scene morphs, and other tracks' audio. Record 4 bars of an evolving scene, then loop and modulate with filter/pitch p-locks. The "OT sampling itself" workflow is one of the deepest performance techniques on the machine.
