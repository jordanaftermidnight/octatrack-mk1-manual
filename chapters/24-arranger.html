The arranger

The Octatrack's song mode. Chains patterns into long-form arrangements with per-row tempo, length, and offset overrides. Eight arrangements per project.

What an arrangement is

An arrangement is a list of rows. Each row plays a specific pattern for a specified duration before advancing to the next row. With careful row design, you can build full-length songs out of sequenced patterns — verse, chorus, bridge, drop, outro — without manually changing patterns during playback.

A project holds 8 arrangements, accessible via FUNCTION + PATTERN. Each arrangement can have up to 256 rows.

Row anatomy

Each row specifies:

FieldFunction
PatternWhich pattern (e.g. A01, B05) to play
LengthHow many bars (or pattern repeats) to stay on this row
OffsetSkip the first N steps of the pattern
BPMOverride project tempo for this row
MuteMute specific tracks for this row
TransposePer-row pitch shift in semitones
SceneForce scene assignments for this row
TrigOptional trigger conditions before advancing

Building an arrangement

  1. Open the arranger page (FUNCTION + PATTERN)
  2. Use UP/DOWN to navigate row by row; LEFT/RIGHT to switch between fields within a row
  3. Knob A selects pattern; B sets length; etc.
  4. Add rows as needed by stepping past the last row
  5. Save with FUNCTION + YES

Activate arrangement playback: from the arrangement page, press PLAY. The OT plays through rows in sequence. To jump to a specific row during playback, navigate to it and press YES.

Per-row overrides

Tempo

The BPM field overrides the project tempo for the duration of that row. Useful for tempo modulation across a song — gradually accelerating drum-and-bass build-ups, or slow-down outros. Setting BPM = 0 inherits from the project.

Mute

The Mute field lets you specify which tracks are muted during the row. Set up base patterns with everything playing, then mute elements per row to create song structure without rewriting patterns.

Transpose

Per-row pitch shift. The OT's notion of "musical key" is at the arrangement row level. Shift +5 semitones for a key change at the chorus, –3 for a darker bridge.

Loops within arrangements

A row can be set to LOOP — it stays active until something triggers it to advance. Useful for live performance where you want a section to vamp until you decide to move on. Trigger advance via the trig keys or via MIDI.

Multiple arrangements per project

With 8 arrangements per project, you can store entire setlists. One arrangement per song; switch arrangements between songs in a set. The 8-arrangement limit is generous for most use cases.

Arranger vs live pattern switching

Arranger is one workflow; manual pattern switching is the other. Most performers use both: arrangements for structured material, manual switching for improvisation. The OT supports interrupt-the- arrangement workflow where you can stop the current row and switch to manual control mid-song.

Tip · row-level scenes

The Scene field lets each row force the crossfader to specific scene assignments. Combined with Mute and Transpose, this means a single arrangement row carries a complete musical context — not just "play pattern A" but "play A with these mutes, this transpose, this scene assignment". An arrangement of 16 rows is effectively a 16-state composition machine.
