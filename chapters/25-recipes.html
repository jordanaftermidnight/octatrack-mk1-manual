Chapter 25

Performance recipes

Workflow patterns for getting useful results out of the Octatrack quickly. Each recipe is a starting framework — adapt to your context.

Recipe — first 90 minutes

Get sound out of the OT, fast

Goal: from boot to a working four-bar sketch in under 90 minutes.

  1. Boot the OT. Verify CF card is installed and formatted.
  2. Create a new project: PROJECT → NEW. Name it.
  3. Drop a few drum samples (kick, snare, hat) into the Audio Pool via the Sample Manager. Assign each to a Flex slot.
  4. On T1, load a Flex machine. Assign the kick slot.
  5. Place trigs on steps 1, 5, 9, 13. You have a four-on-the- floor kick.
  6. On T2, Flex machine, assign snare. Trigs on 5 and 13.
  7. On T3, Flex machine, assign hi-hat. Trigs on 3, 7, 11, 15 (or 16 trigs with conditional probability).
  8. Press PLAY. You have a beat.
  9. Spend the next 80 minutes adjusting volumes, pitches, and adding p-locks. Don't worry about perfection.

The point is to get audio happening fast and iterate from there. The OT rewards iteration.

Recipe — beat slicing

Turn a drum loop into 16 playable hits

  1. Load a 1-bar drum loop into a Flex slot.
  2. Open the audio editor (chapter 20). Go to the SLICE page.
  3. Auto-slice with grid = 16. The loop is now divided into 16 equal slices.
  4. Save. Return to the track.
  5. Switch to SLICES trig mode (FUNCTION + UP/DOWN).
  6. Each trig key now plays a different slice. Build new patterns by hitting trigs in any order.
  7. Switch back to TRACKS mode. Place trigs and p-lock the STRT (slice number) parameter to choose which slices fire on which steps.

Two minutes of setup yields a slice-able drum kit playable from any pattern.

Recipe — DJ-style transition

Smooth pattern transition via the crossfader

  1. Pattern A: your current section.
  2. Pattern B (in same part): your destination section. Same tempo.
  3. Configure scene A (fader fully left): all tracks at full volume, all FX at base settings.
  4. Configure scene B (fader fully right): T1 (kick) muted, T2 (bass) cutoff at 30, T3 (snare) volume at 0, FX1 reverb mix at 100.
  5. While pattern A plays, slide fader from A to B over 4 bars. Pattern A "filters out" and reverbs up.
  6. At the end of the slide, switch to pattern B. The crossfader is now at B end of pattern B's scene mapping.
  7. Slide back to A end over 4 bars. Pattern B "fades in" with full energy.

The transition reads as a 16-bar build-and-drop without ever manually crossfading two tracks.

Recipe — live looping with Pickup

One-take loop performance

  1. T1 = Pickup (Master), source = Input AB, length = 4 bars.
  2. T2 = Pickup (Slave), source = Input AB.
  3. T3 = Flex playback (drums) — loop or pattern.
  4. Plug guitar/synth into Input A.
  5. Place a one-shot trig on T1 step 1.
  6. Press PLAY. Drums start. T1 starts recording at step 1.
  7. Play 4 bars of guitar. T1 captures, then loops.
  8. Place a one-shot trig on T2 step 1 of the next pattern cycle.
  9. Play 4 bars of an overdub layer (different chord / harmony). T2 captures, loops.
  10. You now have drums + 2 layers of looping. Apply scenes for mute/unmute on the loops; build energy curves.

Recipe — generative drum pattern

Conditional trigs + LFO modulation = pattern that mutates

  1. Build a base drum pattern: kick on 1/5/9/13, snare on 5/13, hat on every step.
  2. Add 8 more "ghost" trigs across the pattern. Set each to condition 50% (probability).
  3. Add 4 more "fill" trigs. Set each to condition FILL.
  4. On the snare track, configure LFO 1: WAVE = random, DEST = STRT, DEPTH = ±20, TRIG mode = TRIG. Each snare hit plays from a random offset within the sample.
  5. On the hi-hat track, configure LFO 2: WAVE = sine, DEST = AMP VOL, DEPTH = ±10, MULT = 4 (one cycle per bar). The hat volume gently breathes.
  6. Press PLAY. The pattern is structurally identical every bar but sonically different — the 50% trigs come and go, the snare sample-start jumps around, the hat volume breathes.
  7. Hit FUNCTION+YES to drop a fill momentarily.

Recipe — hub workflow (OT + external gear)

OT as the central brain of a small live setup

  1. OT is master. Configure CLOCK SEND = ON, TRANSPORT SEND = ON.
  2. Connect OT MIDI OUT → external synth (e.g. Digitone) MIDI IN.
  3. Connect synth audio out → OT Input AB.
  4. On the OT, set up a MIDI track sequencing the synth (chapter 22). P-lock notes per step.
  5. Set up an OT audio track with a Thru machine on Input AB — the synth's audio runs through the OT's FX.
  6. The OT now sequences the synth AND processes its audio. Apply scenes to morph FX on the synth's audio. Use the OT's recorder to capture portions of the synth's output as samples.

Result: one OT centralises sequencing, audio routing, FX processing, and live capture for an entire small rig.

Recipe — randomisation as composition tool

Use randomisation to escape "blank pattern paralysis"

  1. Start with an empty pattern.
  2. FUNCTION + RANDOM (the trig key labelled RANDOM in conditions) populates the pattern with random trigs.
  3. Listen for what works. Keep the trigs you like; remove the rest.
  4. Random-LFO on the SLOT parameter cycles through samples per step — yielding chaotic but surprisingly musical beats.
  5. Use 50% conditional trigs for organic-feeling variation.
  6. Iterate. Randomisation is a starting point; your taste is the editor.

Recipe — the dub send

Throw any sound into delay/reverb at any moment

  1. T8 = Master machine with FX1 = Delay, FX2 = Reverb.
  2. Configure each track's CUE level: 0 by default.
  3. Live performance: hold CUE + a TRACK button to send that track's audio to the CUE bus, which (via the Master machine's FX configuration) routes through delay and reverb.
  4. Releasing the CUE+TRACK combo cuts the send. The delay/reverb tails decay naturally.

Effectively a one-button "dub throw" on any track at any time.
