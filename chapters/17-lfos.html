LFOs & the LFO designer

Three LFOs per track, plus a full custom-shape designer. The Octatrack's LFO system is one of the most flexible in any hardware sampler — once you find the designer.

Three LFOs per track

Every audio track has three independent LFOs. Each one can target almost any parameter the track exposes — pitch, filter cutoff, FX parameters, even sample slot.

The LFO page (button to the right of AMP) shows all three at once:

KnobParamFunction
ASPDSpeed (rate)
BMULTSpeed multiplier (sync to bars/steps)
CFADFade-in / fade-out time
DDESTDestination parameter
EWAVEWaveform (sine, tri, saw, square, exp, ramp, random, custom)
FDEPTHModulation depth (positive or negative)

Switching between LFO 1, 2, 3

Use the UP/DOWN arrow keys (within the LFO page) to switch between the three LFOs. The LCD title bar shows which is active.

SPD vs MULT

SPD sets the raw rate; MULT modifies it to align with musical divisions. MULT × SPD = effective rate. Common MULT values:

  • MULT = 1 — free-running, SPD is the rate
  • MULT = 32, 16, 8, 4 — bar-synced (full bar, half, quarter, eighth)
  • MULT = 64, 128 — multi-bar cycles

Trig modes

LFOs can be set to:

  • FREE — runs continuously, ignores trigs
  • TRIG — restarts on every trig
  • HOLD — runs while a trig is active, holds at last value when released
  • ONE — fires once per trig and stops
  • HALF — fires for half a cycle per trig

TRIG mode is the default. ONE is useful for envelope-like LFOs — e.g. a sawtooth ramping down once per trig acts as a decay envelope.

LFO designer (custom waveforms)

Hold FUNCTION + LFO to enter the LFO designer. Here you can draw custom waveforms with up to 16 points per cycle.

How the designer works

  • The screen shows a 16-step horizontal grid
  • Each step has a value (the LFO's amplitude at that point)
  • Move the cursor with LEFT/RIGHT arrows; adjust the value with knob A or UP/DOWN
  • Set INTERP between points (knob B) — linear interpolation, smooth curves, or step-quantised
  • Press YES to save the custom shape; the LFO can then select it via the WAVE parameter

Why custom LFO shapes matter

Built-in waveforms (sine, square, etc.) cover basic modulation. Custom shapes unlock:

  • Step-locked sequences — a 16-step LFO with each step at a different value becomes a CV sequence in software form
  • Drum-machine-style accents — a 16-step LFO modulating volume with peaks every 4 steps
  • Rhythmic filter sweeps — uneven sweep curves that drum patterns can lock to
  • Pseudo-arpeggiators — a 16-step LFO on pitch cycling through scale degrees

The interpolation modes

  • LIN — linear interpolation between points (creates ramps)
  • EXP — exponential curves (smoother, "musical" feel)
  • S/H — step-quantised (no interpolation, creates stepped sequences)

LFO destinations

The DEST parameter sets which parameter the LFO modulates. Almost every parameter on PLAYBACK, AMP, FX1, and FX2 is available. Some common targets:

  • PLAYBACK STRT — random sample-start jumps (with random LFO)
  • FX1 cutoff — classic auto-wah
  • FX2 delay time — pitch-warping delay
  • AMP VOL — tremolo
  • SLOT — sample-slot rotation (random LFO cycles through slots)

LFOs combined with p-locks

LFOs are part-level (set on the part) but can be p-locked per step. A step can lock LFO speed, depth, or destination — effectively retriggering a different LFO behaviour for one trig.

Combined with conditional trigs and step-locked LFO restarts, the OT's modulation system is closer to a generative-music engine than a static synth: same pattern, different sound every cycle.

Hidden · LFO as envelope

Set LFO1 to ONE trig mode, MULT = 1, SPD adjusted to taste, WAVE = exp ramp down, DEPTH = positive. Target a parameter and the LFO becomes a one-shot decay envelope. The OT's LFO system can substitute for a second amplitude envelope, a pitch envelope, or a filter envelope — none of which the unit officially has.
