MIDI tracks

Eight MIDI tracks running in parallel with the audio tracks. The OT can sequence external gear with the same trig + p-lock paradigm — including LFOs, scenes, and conditional trigs.

Activating MIDI mode

Press the MIDI button (callout 11). The button's LED lights, and the trig row, parameter pages, and encoders now address MIDI tracks instead of audio tracks. Press again to return to audio.

The eight MIDI tracks are independent of the eight audio tracks. They share the trig keys but have their own parts, patterns, parameters, and scenes.

Per-MIDI-track configuration

On the MIDI Setup page (FUNCTION + PLAYBACK while in MIDI mode):

ParamFunction
CHANMIDI channel 1–16
OUTOutput port (MIDI Out, USB-MIDI not supported, internal channel routing)
BANKProgram/Bank Select
PROGProgram Change
NOTEBase note (MIDI note number)
VELDefault velocity
LENDefault note length

The five MIDI parameter pages

In MIDI mode, the parameter row buttons map to:

  • NOTE — the playback page for MIDI: note, velocity, length, channel-related params
  • ARP — arpeggiator settings
  • LFO — same three-LFO system as audio, targets MIDI parameters
  • CTRL 1 — first user-assignable CC page (8 CC targets)
  • CTRL 2 — second user-assignable CC page (8 CC targets)

CTRL 1 and CTRL 2 each let you assign 8 CC numbers to the encoders — 16 CCs total per MIDI track, each with full p-lock and LFO capability. This makes the OT a deeply capable MIDI controller, not just a sequencer.

The arpeggiator

The ARP page activates a per-track arpeggiator. Settings:

  • Mode (up, down, alternate, random, polyphonic)
  • Speed (multiplier of project tempo)
  • Octave range
  • Length (how many cycles before stopping)
  • Probability (each arp note can have a percentage chance)

The arpeggiator is rare in performance samplers and is a strong point of the OT's MIDI implementation. Combined with note p-locks you can sequence arps that change pattern and rate per step.

MIDI sequencing examples

Recipe — sequence external synth

Set MIDI track 1's CHAN to match the synth's receive channel. Place trigs on the trig row. Each trig fires a note at the base pitch (NOTE param). P-lock pitch per step for a melodic sequence. The synth plays the OT's notes — you've turned the OT into an external sequencer with full Octatrack-style p-lock capabilities.

Recipe — CC modulation alongside notes

On the same MIDI track, configure CTRL 1 with the synth's filter CC. P-lock the CC per step alongside note p-locks. Now your sequenced notes carry filter modulation per step — no need for the synth to handle the modulation internally.

MIDI receive

The OT also receives MIDI on the IN port. Common uses:

  • External clock (PROJECT → SYNC SOURCE = MIDI)
  • Note-triggering audio tracks (MIDI note ON triggers a sample playback)
  • CC remote control of OT parameters (assignable in PROJECT → MIDI MAP)

The MIDI MAP lets you assign external CCs to OT parameters (filter cutoff, FX values, scene morphs, etc.). Combined with an external controller (or Push, or another sequencer) the OT becomes deeply remote-controllable.

TurboMIDI

Between two Elektron devices, the MIDI cables negotiate a higher baud rate (TurboMIDI), reducing latency on dense CC streams. This is automatic — connect Elektron to Elektron, get TurboMIDI. With non-Elektron gear, falls back to standard MIDI baud (no impact).

Useful in practice for sequencing other Elektron units: Digitone, Analog Rytm, Digitakt — all benefit from TurboMIDI when sequenced from the OT.
