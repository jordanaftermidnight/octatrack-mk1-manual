Eight tracks, master, signal flow

How audio actually moves through the Octatrack. The signal path is simple in outline and becomes powerful through the combinatorics of what each track can do.

Eight audio tracks

The Octatrack has eight audio tracks, T1 through T8. Each track holds one machine — a chunk of sample-playback or audio-routing logic — and processes that machine's output through a fixed signal chain: PLAYBACK → AMP → FX1 → FX2 → mixer routing.

All eight tracks are equal in capability except T8, which has a special role: when its assigned machine is the Master machine, T8 becomes a master-bus processor (chapter 16). Otherwise T8 is a normal audio track.

Alongside the audio tracks is a parallel set of eight MIDI tracks. MIDI tracks share the same trig keys when MIDI mode is active (chapter 11 of Part IX); they don't process audio.

Per-track signal chain

1. Source

The track's machine produces audio. For Static and Flex machines, this is the active sample slot's audio, played at the position determined by current trigs and parameter locks. For Thru machines it's the live audio from a chosen rear-panel input pair. For Pickup machines it's the contents of that track's recorder buffer.

2. PLAYBACK page

Pitch, sample start, length, loop, time-stretching, slice selection. These shape what portion of the source plays and how. See chapter 21.

3. AMP page

Amplitude envelope (attack, hold, release), volume, balance (pan). Every Octatrack track has an AMP envelope; a flat envelope is just "track plays at full volume". See chapter 22.

4. FX1, FX2

Two effect slots per track. Each slot can hold any one of the available effects (filter, EQ, delay, reverb, chorus, compressor, phaser, lo-fi, etc.). FX1 runs before FX2; the order is fixed. Choosing which effect lives in each slot is done on the FX page's Setup screen.

5. Mixer routing

After FX2, the track's audio is routed to one or more of:

  • MAIN bus — the master stereo output
  • CUE bus — pre-listen / Studio Mode second pair
  • Both, with independent levels

Routing is set on the Mixer page. The MAIN bus is summed across all eight tracks; if T8 is loaded with a Master machine, that sum runs through T8's PLAYBACK / AMP / FX1 / FX2 chain before reaching the rear-panel MAIN outputs.

The Master track quirk

When T8's machine is set to Master, T8 stops being a normal track and becomes a master-bus effects insert. Its sample slot is ignored; its trigs are ignored; its sequencer page does nothing audible. What it does have is its FX1 and FX2 slots, which now process the entire MAIN-bus sum.

This is how you get a master EQ or master compressor on the Octatrack: load the Master machine on T8, then put a Compressor on its FX1 and a final EQ on its FX2. Note that you lose T8 as a sound-producing track when you do this — eight becomes seven.

The crossfader is not in the chain

The crossfader morphs parameter values between two snapshots (Scene A and Scene B). It does not gate audio or sit in any signal path. When you move the fader, every parameter that has a scene lock interpolates between the A value and the B value. The audio signal flow above is the same whether the fader is at A, at B, or anywhere between.

A useful corollary: if you scene-lock a track's level to 0 at Scene A and to maximum at Scene B, the fader will appear to fade that track in or out. But the audio still flows through the same chain — the level parameter is what's changing.

What this means in practice

With one machine per track, you build composite voices through routing (Neighbor machines that read the previous track's output — see chapter 16) or by parallel arrangement (drum loop on T1, bass on T2, atmosphere on T3, etc.). Eight tracks is more than it sounds when each track has its own two effect slots, its own LFO, and full parameter-lock control per step.

Tip · the FX1/FX2 ordering hack

Because FX1 always runs before FX2, the choice of which effect goes where matters. A reverb on FX1 followed by a filter on FX2 sounds very different from filter→reverb. When you find a combination that doesn't sound right, try swapping the slots before swapping the effects.
