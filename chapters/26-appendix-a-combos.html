Appendix A

Button combo encyclopedia

Every documented and hidden key combination on the Octatrack MK1, organised by primary key. Print-friendly.

Format

Each combo is shown as FUNCTION+KEY. Some combos require multiple steps (hold A, then press B). "FN" is shorthand for FUNCTION.

Transport

ComboAction
PLAYPlay / pause sequencer
STOPStop sequencer
STOP+STOPStop and rewind to step 1
FN+STOPStop all Pickup tracks immediately
REC+PLAYLive Record (unquantised)
FN+REC+PLAYLive Record (quantised)
REC alone (in record mode)Exit Live Record without stopping

Trigs and editing

ComboAction
TRIGToggle trig on/off
TRIG hold + knobSet parameter lock for that step
TRIG hold + LEFT/RIGHTSet microtiming
TRIG hold + LEVEL knobSet conditional trig
TRIG hold + NOClear all locks on that step
TRIG hold + knob pressClear that parameter's lock on that step
FN+TRIGTrigless lock (parameter changes without re-trigger)
YES+TRIGPreview step (with locks), to MAIN
CUE+TRIGPreview step (with locks), to CUE
FN+NO (in Live Record)Erase trigs as playhead passes

Track selection

ComboAction
T1–T8Select that track
T1–T8 holdMute that track (momentary)
FN+T1–T8Toggle track mute
CUE+T1–T8Toggle track CUE send
Two T buttons togetherSolo (mute everything else)

Trig modes

ComboAction
FN+UP/DOWNCycle through trig modes (TRACKS / CHROMATIC / SLICES / SLOTS / QUICK MUTE / DELAY CONTROL)
UP/DOWN (in CHROMATIC)Octave shift
LEFT/RIGHT (in SLICES)Page through slice ranges

Patterns and banks

ComboAction
PATTERN+TRIG 1–16Switch to that pattern within the bank
BANK+TRIG 1–16Switch to that bank (A–P)
FN+PATTERNOpen arranger
FN+BANKEdit pattern (audio editor for current track)
FN+YES (on pattern page)Save pattern

Parts and scenes

ComboAction
SCENE A+TRIGAssign that scene number to fader's A end
SCENE B+TRIGAssign that scene number to fader's B end
SCENE A+knobSet scene-locked value for active parameter
FN+SCENE AMute scene A's effect (clear A assignment temporarily)
FN+YES (on part page)Save part
MIDI hold (PART)Open part menu (part list, save, load, copy)

Cheat sheet — top 10 most-used

  1. TRIG hold + knob → p-lock
  2. FN+TRIG → trigless lock
  3. YES+TRIG → preview step
  4. REC+PLAY → Live Record
  5. FN+UP/DOWN → cycle trig modes
  6. SCENE A+knob → scene-lock parameter
  7. FN+YES → save current item
  8. FN+NO → reload
  9. BANK+TRIG → switch bank
  10. PATTERN+TRIG → switch pattern
